# PostgreSQL

## 預設資料庫(database)

PostgreSQL安裝好後，會先創建postgres，template0和template1三個空資料庫。&#x20;

* postgres用作登錄時的預設資料庫，也包含了資料庫的設定表格。
* template0和template1是範本資料庫，用於創建其他資料庫。&#x20;
* 資料庫的資訊存儲在pg\_database表中，也可以在psql下使用\list命令查看。
